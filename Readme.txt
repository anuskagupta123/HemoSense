Smart Bridge – AnemiaSense
Machine Learning Model Deployment using Flask

Project Structure:
- forms/      → input forms (optional)
- static/     → CSS styling
- templates/  → HTML pages
- model.pkl   → ML model
- app.py      → Flask backend

Run Project:
1. Install dependencies:
   pip install flask numpy scikit-learn

2. Start server:
   python app.py

3. Open browser:
   http://127.0.0.1:5000/
